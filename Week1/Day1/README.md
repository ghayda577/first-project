# Data Science Environment Setup

## Overview
This repository contains the setup and initial testing of a Data Science environment using Python.  
The purpose of this lab was to create a virtual environment, install the main data science libraries, and verify that everything is working correctly.

This notebook was only an initial experiment with Jupyter Notebook to understand the workflow and test the environment setup before starting more advanced data analysis tasks.

## Tools and Libraries Used
- Python Virtual Environment (venv)
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib

## Steps Completed
1. Created and activated a virtual environment.
2. Installed the required data science libraries:
   - NumPy
   - Pandas
   - Matplotlib
   - Jupyter
3. Created a Jupyter Notebook containing:
   - Markdown cells for explanations.
   - Code cells for execution and testing.
4. Checked the installed versions of all libraries to confirm successful installation.
5. Generated a `requirements.txt` file using `pip freeze`.
6. Initialized a Git repository and uploaded the project files to GitHub.

## Purpose
The main goal of this lab was to become familiar with the data science development environment, Jupyter Notebook structure, and basic project organization using Git and GitHub.

## Note
This notebook represents a first hands-on trial with Jupyter Notebook and environment setup. It is intended for learning and testing purposes before moving on to real data analysis projects.
