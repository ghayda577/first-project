# NumPy Fundamentals Exercises

## Overview
This repository contains hands-on exercises to practice the fundamentals of NumPy, one of the most important libraries used in Data Science and Machine Learning.

In this lab, we started working with arrays and explored basic NumPy operations such as creating arrays, checking array properties, slicing, filtering data using conditions, and applying broadcasting.

An additional exercise was added to practice and strengthen the understanding of NumPy concepts through more experimentation with arrays and operations.

## Concepts Practiced
- Creating and manipulating 2D arrays.
- Understanding array shape and data types (`shape` and `dtype`).
- Extracting specific rows and columns using slicing.
- Applying boolean masks to filter array values based on conditions.
- Understanding broadcasting and performing operations between arrays with different dimensions.
- Verifying results manually to ensure correct calculations.

## Tasks Completed

2. Used slicing techniques to extract:

   - The last row of the array.

3. Applied a boolean mask to select all elements greater than the array mean.

5. Added an extra exercise to gain more practice with NumPy array operations and improve understanding of the library.

## Purpose
The purpose of this lab was to build a foundation in NumPy and understand how arrays are created, accessed, filtered, and manipulated, which are essential skills for future Data Science tasks.
