# Python Fundamentals Drill

## Overview
This repository contains the first hands-on practice with Python fundamentals.  
The goal of this lab was to start building a strong foundation in Python programming by practicing basic concepts and understanding how Python code is structured.

During this exercise, we worked on writing functions, using list comprehensions, creating classes, and documenting code using Markdown cells in Jupyter Notebook.

## Concepts Practiced
- Creating and using Python functions.
- Working with lists and performing basic statistical operations.
- Using list comprehensions as a shorter and cleaner way to write loops.
- Understanding Object-Oriented Programming (OOP) basics by creating a simple class.
- Adding Markdown explanations to document the purpose of each code cell.

## Tasks Completed
1. Created a function that receives a list of numbers and returns:
   - Mean
   - Minimum value
   - Maximum value  
   in a dictionary format.

2. Converted a traditional for-loop used for filtering even numbers into a single list comprehension.

3. Designed a simple Python class representing a data record with:
   - Attributes to store data.
   - A method to display or process the record.

4. Added Markdown cells to explain what each code section does and why it is used.

## Purpose
The purpose of this lab was to become familiar with Python basics and practice writing clean, organized, and understandable code before moving to more advanced topics in Data Science and Machine Learning.
